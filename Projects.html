Introduction Projects Portfolio

Projects

    1. Living+

    Living+ is a web application that helps users find roommates based on their preferences. Users can create an account, fill out a questionnaire, and get matched with potential roommates. The app also allows users to search for available rooms and filter results based on location, price, and other criteria.

    2. Forkify

    Forkify is a recipe search engine that allows users to search for recipes based on ingredients, diet, and other criteria. Users can save recipes, create shopping lists, and view detailed instructions for each recipe. The app also includes a feature that allows users to adjust recipe quantities based on the number of servings.

    3. Fresh-N-Fast

    Fresh-N-Fast is an online grocery store that offers a wide range of fresh produce, dairy products, and other groceries. Users can browse through different categories, add items to their cart, and place orders for home delivery. The app also provides features like order tracking, scheduled deliveries, and personalized recommendations based on user preferences.

    4. Pacman

    Pacman is a classic arcade game where players control a yellow character that moves around a maze eating dots and avoiding ghosts. The game features multiple levels, power-ups, and bonus items that enhance gameplay. Players can compete for high scores and challenge friends to beat their records.

    5. FHS

    FHS (Falcon House School) is a comprehensive school website that provides information about the school's programs, faculty, and events. The website includes sections for admissions, academics, extracurricular activities, and contact information. It also features an online portal for students and parents to access grades, schedules, and other important resources.
